# 倒排索引
传统的我们的检索是通过文章，逐个遍历找到对应关键词的位置。而倒排索引，是通过分词策略，形成了词和文章的映射关系表，这种词典+映射表即为倒排索引。有了倒排索引，就能实现o（1）时间复杂度 的效率检索文章了，极大的提高了检索效率。

倒排索引就是关键词到文档 ID 的映射，每个关键词都对应着一系列的文件，这些文件中都出现了关键词。
# 分词器的配置
```
{
    "analyzer": "standard",
    "text": "哪吒之魔童降世"
}
哪 吒 之 魔 童 降 世
```
```
{
    "analyzer": "ik_max_word",
    "text": "哪吒之魔童降世"
}
哪吒 之 魔 童 降世
```
